* SumOfArray Project

*** Overview:

This is a simple Java console application that allows the user to:

1. Define the size of an array
2. Fill the array with integer values
3. Display the array
4. Calculate the sum of all elements in the array

*** Covered:

1. Arrays in Java
2. Scanner Input Handling
3. Exception Handling
4. Loops
5. Input Validation
6. Methods in Java
7. Static vs Non-Static Methods
8. Arrays Utility Class
